# 🐦 FlyingBird Game

A fun and interactive **Flying Bird Game** developed using **HTML, CSS, and JavaScript**.  
Experience smooth **animations and transitions**, challenge your skills, and track your score as you navigate the bird through obstacles!  

---

## ✨ Features

- 🕹️ **Start, Pause & Play** options  
- 🎯 **Scoreboard** to track your progress  
- 🔄 **Restart button** after crashing  
- 🌟 Smooth **animations and transitions**  
- 📱 Fully playable in web browsers  

---

## 🚀 How to Run

1️⃣ **Download and Extract**  
Download the game files and extract them to a folder.  

2️⃣ **Open VS Code**  
Launch **Visual Studio Code**.  

3️⃣ **Open Folder**  
Go to `File > Open Folder...` and select the extracted game folder.  

4️⃣ **Open HTML File**  
Click on the HTML file in the Explorer sidebar.  

5️⃣ **Start Live Server**  
Right-click the HTML file and select `Open with Live Server`.  

6️⃣ **Play and Enjoy**  
The game will open in your browser. Fly the bird and enjoy!
